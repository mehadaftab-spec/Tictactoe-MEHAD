# Tic Tac Toe (Godot Web Build)

## Overview
This project is a browser-playable Tic Tac Toe game developed using the Godot Engine. It implements the standard 3x3 grid gameplay with a simple user interface and has been exported for the web and deployed using GitHub Pages.

## Features
- Two-player turn-based Tic Tac Toe
- 3x3 grid with win and draw detection
- Custom images used in place of default X and O symbols
- Interactive buttons for gameplay control (restart, menu, etc.)
- Web deployment for direct play in a browser

##Tech used
- Godot Engine (GDScript)
- HTML5/Web export
- GitHub Pages for hosting

## How to Play
1. Open the game using the deployed link.
2. Players take turns clicking on empty cells.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Use the on-screen controls to restart or navigate.

##Deployment
The game is exported using Godot’s Web (HTML5) export and hosted via GitHub Pages. The required files include:
- index.html
- index.js
- index.wasm
- index.pck
